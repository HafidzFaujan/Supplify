SAW - Pemilihan Supplier untuk Toko Sparepart

Deskripsi:
Aplikasi web sederhana (HTML + JavaScript) untuk mendemonstrasikan metode Simple Additive Weighting (SAW) 
pada studi kasus pemilihan supplier sparepart.

Cara pakai:
1. Buka index.html di browser (cukup klik file atau jalankan lewat HTTP server).
2. Klik "Load Sample Data" untuk memuat data contoh.
3. Ubah bobot jika perlu (jumlah disarankan = 1).
4. Klik "Compute SAW" untuk melihat ranking supplier.
5. Anda dapat mengunggah CSV dengan format header:
   name,price,quality,delivery_time,min_order,rating

File dalam paket:
- index.html
- styles.css
- app.js
- suppliers.csv (sample)
- README.md
